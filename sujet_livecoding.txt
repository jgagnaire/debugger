
Projet GueulDeBoua:

le but de notre livecoding est le recode d'un debugger avec les fonctionnalités suivantes :
-> attachement à un process
-> set de breakpoint
-> run du process
-> stop au breakpoint
-> affichage et modification des registres
-> afficher la stack sur demande
-> modification de la valeur des variables pendant le debug

programme qui se lance avec un binaire en paramètre, et qui recoit les commandes suivantes dans un prompt :
-> run
-> quit
-> set
-> break
-> delete
-> print
-> printreg
-> setreg
-> backtrace
